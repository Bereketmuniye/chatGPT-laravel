
# Chat GPT Laravel

I built a Chat GPT App that allows you to... using Laravel... In 5 minutes.

## Resources

https://github.com/openai-php/client

## Requirements

- PHP v8.1
- Composer v2.5.5
- Web Server (HTTPS)

## Setup

- `composer create-project laravel/laravel chatGPT - laravel`
- `cd chatGPT - laravel`
- `php artisan make:controller ChatController`
